## Install Redis on Windows 10

## cmd  : redis-cli

DATA TYPES:
Hashes
Lists
Sets
Sorted Sets


Returns All Keys : keys *

Set : set keyName value
		Ex: set name kamal

Get : get keyName
    Ex: get name

Delete : del  keyName
		Ex: del name

Delete All Keys : flushall
